SetDisplayMode

SetDisplayMode(mixed zoom [, string layout])

Descripció

Defineix com es mostrarà el document al visor. El nivell de zoom pot ser: les pàgines es poden mostrar senceres a la pantalla, acupar l'ample total de la finestra, usar el tamany real, proporcionar per un factor de zoom o usar el predeterminat del visor (configurat al menú de preferències de l'Acrobat). El format de la pàgina es pot especificar també: una cada cop, mostra contínua, dues columnes o el predeterminat del visor.
Per defecte, el documents usen el mode d'ample total amb mostra contínua.

Paràmetres

zoom
El zoom a usar. Pot ser un dels valors de cadena següents:
  • fullpage: mostra la pàgina sencera a la pantalla
  • fullwidth: usa l'ample màxim de la pantalla
  • real: usa el tamany real (equival a zomm del 100%)
  • default: usa el mode predeterminat del visor
o bé un número indicant el factor de zoom a usar.
layout
El format de pàgina. Els valors possibles són:
  • single: mostra una pàgina cada cop
  • continuous: mostra dues pàgines contínues
  • two: mostra dues pàgines en dues columnes
  • default: usa el mode predeterminat del visor
El valor per defecte és continuous.
