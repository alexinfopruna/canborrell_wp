Image

Image(string file [, float x [, float y [, float w [, float h [, string type [, mixed link]]]]]])

Descripció

Posa una imatge a la pàgina. S'hi ha de donar l'angle superior esquerre. Les dimensions es poden especificar de vàries maneres:
  • ample i alt explícit (expressat en unitats d'usuari)
  • una dimensió explícita, l'altra calculada automàticament mantenint les proporcions originals
  • sense dimensions explícites, llavors la imatge es posa a 72 dpi (punt per polzada)
Se suporten els formats JPEG, PNG i GIF. Cal tenir instal·lada l'extensió GD de PHP per als GIF.

De JPEG s'admeten totes les variants:
  • escala de grisos
  • colors vertaders (24 bits)
  • CMYK [Cyan-Magenta-Yellow-Black o Blau-Roig-Groc-Negre] (32 bits)
De PNG s'admeten:
  • escala de grisos de 8 bits (256 nivells) com a molt
  • colors indexats
  • colors vertaders (24 bits)
però no:
  • Entrellaçat
  • canal Alpha
De GIF, si és un GIF animat, només es mostra el primer fotograma.

Si es defineix un color transparent, es tindrà en compte.
El format es pot especificar explícitament o deduir-lo de l'extensió del fitxer.
És possible posar-hi un enllaç.

Comentari: si una imatge s'usa diversos cops, nomes s'hi incrustarà una còpia.

Paràmetres

file
Nom del fitxer que conté la imatge.
x
Abscissa de l'angle superior esquerre. Si no s'especifica o és null, s'usa l'abscissa actual.
y
Ordenada de l'angle superior esquerre. Si no s'especifica o és null, s'usa l'ordenada actual; a més, s'activa abans un salt de pàgina si cal (si s'ha habilitat el salt de pàgina automàtic) i, després de la crida, l'ordenada actual se situa al final de la pàgina.
w
Ample de la imatge a la pàgina. Si no se n'especifica o és zero, es calcula automàticament.
h
Alçada de la imatge a la pàgina. Si no se n'especifica o és zero, es calcula automàticament.
type
Format d'imatge. Els valors possibles són (en majúscula o minúscula): JPG, JPEG, PNG i GIF. Si no se n'especifica, es dedueix de l'extensió del fitxer.
link
URL o identificador tornat per AddLink().
